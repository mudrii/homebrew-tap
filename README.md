# Homebrew Tap

Homebrew formulae for [OpenClaw](https://github.com/mudrii/openclaw-dashboard) tools.

## Install

```sh
brew tap mudrii/tap
brew install openclaw-dashboard
```

Or directly:

```sh
brew install mudrii/tap/openclaw-dashboard
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| `openclaw-dashboard` | Real-time monitoring dashboard for OpenClaw AI gateway |
